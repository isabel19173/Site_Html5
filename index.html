Web Design

1. O que é Web Design e qual sua relação com Front-End?

Web Design é o planejamento visual de interfaces digitais, focado em estética e experiência do usuário, utilizando ferramentas como Figma ou Adobe XD.Enquanto o Front-End é o desenvolvimento da parte visual e interativa usando HTML, CSS e JavaScript. A relação do Web Design com Front-End é traduz o design criado pelo Web Designer em código funcional.

2. Cite três princípios de design que melhoram a experiência do usuário:

  1. Hierarquia visual;
  2. Contraste e alinhamento;
  3. Consistência e acessibilidade.

3. Qual a diferença entre Wireframe e Protótipo?

  • Wireframe: Esboço estrutural simples, focado em layout e funcionalidade, sem elementos visuais finais.
  • Protótipo: Versão interativa avançada que simula a experiência real do usuário com cliques e transições.

4. Liste três ferramentas usadas para criar protótipos:

  1. Figma;
  2. Adobe XD;
  3. Marvel App.

HTML5 e Semântica

5. Explique a função da tag <header> e dê um exemplo de uso.

A tag <header> representa o cabeçalho de uma página ou seção. Geralmente contém logo, menu, título ou informações introdutórias.

 
                    <header>
                    <h1>Título</h1>
                    <nav>
                    <href="#">Home</a>
                    <a href="#">Contato</a>
                    </nav>
                    </header>

6. Qual a diferença entre uma tag em bloco e uma tag inline?

  • Bloco: Ocupam toda a largura disponível (<div>, <p>, <h1>).
  • Inline: Ocupam apenas o espaço do conteúdo (<span>,<a>, <strong>).
  • Importante: Nunca use elementos em bloco dentro de inline.

CSS

7. Como aplicar CSS externo em um documento HTML?

O CSS externo é feito em um arquivo separado com extensão .css (estilo.css) Ele é incluído no HTML através da tag <link>: 


                <link rel="stylesheet" href="style.css" >

8. Cite três propriedades CSS comuns e sua função.

  • color: Define a cor do texto.
  • background-color: Define a cor de fundo.
  • font-size Define o tamanho da fonte. (Outras opções incluem margin e padding).

JavaScript

9. O que é DOM e como o JavaScript interage com ele?

DOM (Document Object Model) é uma representação do HTML como uma árvore de objetos que o JavaScript pode acessar e modificar.
Com isso o JS pode:

  • alterar textos,
  • mudar cores,
  • criar elementos,
  • reagir a eventos (cliques, digitação).
